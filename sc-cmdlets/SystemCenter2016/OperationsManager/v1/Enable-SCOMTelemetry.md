# Enable-SCOMTelemetry

## SYNOPSIS
Enables collection of Operations Manager diagnostic data.

## SYNTAX

```
Enable-SCOMTelemetry [-SCSession <Connection[]>] [-ComputerName <String[]>] [-Credential <PSCredential>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Enable-SCOMTelemetry** cmdlet enables collection of System Center 2016 - Operations Manager diagnostic and usage patterns data by Microsoft.

## PARAMETERS

### -ComputerName
Specifies an array of names of computers.
The cmdlet establishes temporary connections with management groups for these computers.
You can use NetBIOS names, IP addresses, or fully qualified domain names (FQDNs).
To specify the local computer, type the computer name, localhost, or a dot (.).
