# Get-SCOMMaintenanceScheduleList

## SYNOPSIS
Gets a list of all Maintenance Schedules for the connected Management Group.

## SYNTAX

```
Get-SCOMMaintenanceScheduleList [-SCSession <Connection[]>] [-ComputerName <String[]>]
 [-Credential <PSCredential>] [<CommonParameters>]
```

## DESCRIPTION
The **Get-SCOMMaintenanceScheduleList** cmdlet gets a list of all Maintenance Schedules for the connected Management Group.

## PARAMETERS

### -ComputerName
Specifies an array of names of computers.
The cmdlet establishes temporary connections with management groups for these computers.
You can use NetBIOS names, IP addresses, or fully qualified domain names (FQDNs).
To specify the local computer, type the computer name, localhost, or a dot (.).The System Center Data Access service must be active on the computer.
If you do not specify a computer, the cmdlet uses the computer for the current management group connection.

### -Credential
Specifies a **PSCredential** object for the management group connection.
To obtain a **PSCredential** object, use the Get-Credential cmdlet.
For more information, type `Get-Help Get-Credential`.If you specify a computer in the *ComputerName* parameter, use an account that has access to that computer.
The default is the current user.
