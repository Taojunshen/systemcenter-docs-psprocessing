# Register-SCAdvisor

## SYNOPSIS
Registers an Operations Manager management group to an Advisor account.

## SYNTAX

```
Register-SCAdvisor [-CertificatePath <String>] [-SettingServiceUrl <String>] [-SCSession <Connection[]>]
 [-ComputerName <String[]>] [-Credential <PSCredential>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Register-SCAdvisor** cmdlet registers a System Center 2016 Operations Manager management group to a System Center Advisor account.

## EXAMPLES

### Example 1: Register a connector
```
PS C:\>Register-SCAdvisor -CertificatePath "C:\RegistrationCert.pfx"
```

This command registers the current management group to an Advisor account.

### -ComputerName
Specifies an array of names of computers.
The cmdlet establishes temporary connections with management groups for these computers.
You can use NetBIOS names, IP addresses, or fully qualified domain names (FQDNs).
To specify the local computer, type the computer name, localhost, or a dot (.).

The System Center Data Access service must be active on the computer.
If you do not specify a computer, the cmdlet uses the computer for the current management group connection.

Specifies an array of **Connection** objects.
