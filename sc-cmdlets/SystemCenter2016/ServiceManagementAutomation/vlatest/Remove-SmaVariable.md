# Remove-SmaVariable

## SYNOPSIS
Deletes a variable from SMA.

## SYNTAX

```
Remove-SmaVariable -Name <String> -WebServiceEndpoint <String> [-Port <Int32>] [-AuthenticationType <String>]
 [-Credential <PSCredential>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Remove-SmaVariable** cmdlet deletes a variable from Service Management Automation (SMA).
Specify the name of the variable, the web service endpoint, and, if necessary, a port number.

## EXAMPLES

### Example 1: Remove a variable
```
PS C:\> Remove-SmaVariable -WebServiceEndpoint https://localhost -Name "MyVariable"
```

This command removes the variable named MyVariable.
