# Remove-SCCloud

## SYNOPSIS
Deletes a private cloud from VMM.

## SYNTAX

```
Remove-SCCloud [-VMMServer <ServerConnection>] [-Cloud] <Cloud> [-JobGroup <Guid>] [-RunAsynchronously]
 [-PROTipID <Guid>] [-JobVariable <String>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Remove-SCCloud** cmdlet deletes a private cloud from Virtual Machine Manager (VMM).

## EXAMPLES

### Example 1: Delete a specified private cloud
```
PS C:\>Get-SCCloud -VMMServer "VMMServer01.Contoso.com -Name "Cloud01" | Remove-SCCloud -Confirm
```

This command gets the private cloud object named Cloud01 on VMMServer01 and uses the pipeline operator to pass the private cloud object to **Remove-SCCloud**, which prompts you to confirm the action before deleting the private cloud.
