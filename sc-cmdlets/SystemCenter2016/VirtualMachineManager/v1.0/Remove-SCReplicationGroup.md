# Remove-SCReplicationGroup

## SYNOPSIS
Removes a replication group.

## SYNTAX

```
Remove-SCReplicationGroup [-ReplicationGroup] <ReplicationGroup> [-Force] [-RunAsynchronously]
 [-PROTipID <Guid>] [-JobVariable <String>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Remove-SCReplicationGroup** cmdlet removes a replication group.
