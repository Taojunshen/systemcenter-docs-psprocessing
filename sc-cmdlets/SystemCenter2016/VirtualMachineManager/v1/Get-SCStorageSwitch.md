# Get-SCStorageSwitch

## SYNOPSIS
Gets a Fibre Channel switch object.

## SYNTAX

### All (Default)
```
Get-SCStorageSwitch [-VMMServer <ServerConnection>] [[-Name] <String>] [-All] [<CommonParameters>]
```

### ID
```
Get-SCStorageSwitch [-VMMServer <ServerConnection>] [[-Name] <String>] [-ID <Guid>] [<CommonParameters>]
```

## DESCRIPTION
The **Get-SCStorageSwitch** cmdlet gets a Fibre Channel switch object.
