# Remove-SCPhysicalComputerNetworkAdapterProfile

## SYNOPSIS
Removes a physical computer network adapter profile object from VMM.

## SYNTAX

```
Remove-SCPhysicalComputerNetworkAdapterProfile
 [-PhysicalComputerNetworkAdapterProfile] <PhysicalComputerNetworkAdapterProfile> [-Force] [-RunAsynchronously]
 [-PROTipID <Guid>] [-JobVariable <String>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Remove-SCPhysicalComputerNetworkAdapterProfile** cmdlet removes physical computer network adapter profile object from Virtual Machine Manager (VMM).
