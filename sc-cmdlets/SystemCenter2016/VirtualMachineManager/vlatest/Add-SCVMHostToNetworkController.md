# Add-SCVMHostToNetworkController

## SYNOPSIS
Adds a virtual machine host to a network controller.

## SYNTAX

```
Add-SCVMHostToNetworkController [-VmHost] <Host> [-VMMServer <ServerConnection>] [-RunAsynchronously]
 [-PROTipID <Guid>] [-JobVariable <String>] [<CommonParameters>]
```

## DESCRIPTION
The **Add-SCVMHostToNetworkController** cmdlet adds a virtual machine host to a network controller in Virtual Machine Manager (VMM).
