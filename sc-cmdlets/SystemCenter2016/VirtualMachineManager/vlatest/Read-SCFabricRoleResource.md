# Read-SCFabricRoleResource

## SYNOPSIS
Reads a fabric role resource.

## SYNTAX

```
Read-SCFabricRoleResource [-VMMServer <ServerConnection>] [-FabricResource] <FabricRoleResource> [-Recursive]
 [-RunAsynchronously] [-PROTipID <Guid>] [-JobVariable <String>] [-OnBehalfOfUser <String>]
 [-OnBehalfOfUserRole <UserRole>] [<CommonParameters>]
```

## DESCRIPTION
The **Read-SCFabricRoleResource** cmdlet reads a fabric role resource in Virtual Machine Manager (VMM).
