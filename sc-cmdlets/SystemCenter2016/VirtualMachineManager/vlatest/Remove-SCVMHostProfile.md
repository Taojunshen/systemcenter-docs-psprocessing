# Remove-SCVMHostProfile

## SYNOPSIS
Removes a host profile object from the VMM database.

## SYNTAX

```
Remove-SCVMHostProfile [-VMHostProfile] <PhysicalComputerProfile> [-Force] [-RunAsynchronously]
 [-PROTipID <Guid>] [-JobVariable <String>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Remove-SCVMHostProfile** cmdlet removes a host profile object from the Virtual Machine Manager (VMM) database.

## EXAMPLES

### Example 1: Remove a specified host profile
```
PS C:\> Get-SCVMHostProfile -Name "HostProfile01" | Remove-SCVMHostProfile
```

This command removes the host profile object named HostProfile01.
