# Set-CloudVmRoleScale

## SYNOPSIS
Updates a cloud virtual machine role scale.

## SYNTAX

```
Set-CloudVmRoleScale [-VmRole] <CloudVmRole> [-VMMServer <ServerConnection>] -VmCount <Int32> [-RunREST]
 [-JobVariable <String>] [-OnBehalfOfUser <String>] [-OnBehalfOfUserRole <UserRole>] [<CommonParameters>]
```

## DESCRIPTION
The **Set-CloudVMRoleScale** cmdlet updates a cloud virtual machine role scale.
