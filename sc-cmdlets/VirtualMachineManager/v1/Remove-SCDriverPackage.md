# Remove-SCDriverPackage

## SYNOPSIS
Removes a driver package object from the VMM library.

## SYNTAX

```
Remove-SCDriverPackage [-DriverPackage] <DriverPackage> [-Force] [-RunAsynchronously] [-PROTipID <Guid>]
 [-JobVariable <String>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## DESCRIPTION
The **Remove-SCDriverPackage** cmdlet removes a driver package object from the Virtual Machine Manager (VMM) library.

## EXAMPLES

### Example 1: Remove a specific driver package
```
PS C:\>Get-SCDriverPackage -PnPID "DRIVER" | Remove-SCDriverPackage -Confirm
```

This command removes the driver package object with the Plug and Play ID of "DRIVER" from the VMM library.
