# Remove-SCNetworkConnection

## SYNOPSIS
Removes a network service connection.

## SYNTAX

```
Remove-SCNetworkConnection [-VMMServer <ServerConnection>]
 [-NetworkServiceConnection] <NetworkServiceConnection> [-Force] [-RunAsynchronously] [-PROTipID <Guid>]
 [-JobVariable <String>] [<CommonParameters>]
```

## DESCRIPTION
The **Remove-SCNetworkConnection** cmdlet removes a network service connection.
